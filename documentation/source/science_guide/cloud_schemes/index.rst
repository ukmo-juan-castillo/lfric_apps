Cloud Schemes available in LFRic
================================

Contained here are descriptions of the cloud-schemes available in the
LFRic atmosphere model.  By "cloud-scheme", we mean a scheme for calculating
the sub-grid cloud-fraction and water-content as a function of the
grid resolved variables.  This requires making some assumptions about
the PDF of sub-grid moisture variability.
